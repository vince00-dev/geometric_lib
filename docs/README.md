# Math formulas

## Circle area

- S = πR²
- Передается число r, возвращается число math.pi _ r _ r (площадь круга)
- Если вызвать функцию area(5), то она вернет 78.53981633974483

Пример использования:

```python
import circle
print(circle.area(5))  # 78.53981633974483
```

## Circle perimeter

- P = 2πR
- Передается число r, возвращается число 2 _ math.pi _ r (периметр круга)
- Если вызвать функцию perimeter(5), то она вернет 31.41592653589793

Пример использования:

```python
import circle
print(circle.perimeter(5))  # 31.41592653589793
```

## Square area

- S = a²
- Передается число a, возвращается число a \* a (площадь квадрата)
- Если вызвать функцию area(4), то она вернет 16

Пример использования:

```python
import square
print(square.area(4))  # 16
```

## Square perimeter

- P = 4a
- Передается число a, возвращается число 4 \* a (периметр квадрата)
- Если вызвать функцию perimeter(4), то она вернет 16

Пример использования:

```python
import square
print(square.perimeter(4))  # 16
```

## Rectangle area

- S = ab
- Передается число a и b, возвращается число a \* b (площадь прямоугольника)
- Если вызвать функцию area(5, 10), то она вернет 50

Пример использования:

```python
import rectangle
print(rectangle.area(5, 10))  # 50
```

## Rectangle perimeter

- P = 2(a+b)
- Передается число a и b, возвращается число 2 \* (a + b) (периметр прямоугольника)
- Если вызвать функцию perimeter(5, 10), то она вернет 30

Пример использования:

```python
import rectangle
print(rectangle.perimeter(5, 10))  # 30
```

## Triangle area

- S = ah/2
- Передается число a и h, возвращается число a \* h / 2 (площадь треугольника)
- Если вызвать функцию area(6, 8), то она вернет 24.0

Пример использования:

```python
import triangle
print(triangle.area(6, 8))  # 24.0
```

## Triangle perimeter

- P = a+b+c
- Передается число a, b и c, возвращается число a + b + c (периметр треугольника)
- Если вызвать функцию perimeter(3, 4, 5), то она вернет 12

Пример использования:

```python
import triangle
print(triangle.perimeter(3, 4, 5))  # 12
```
